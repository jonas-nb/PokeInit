
# Pokeinit

Contains a pokedex with 151 first generation pokemons.
## Running locally

clone the project

```bash
  git clone https://github.com/jonas-nb/PokeInit
```

Enter the project directory

```bash
  cd pokeinit
```

install the dependencies

```bash
  npm install
```

start the server

```bash
  npm run dev
```


## Stack utilizada

**Front-end:** React, TailwindCSS, Vite, React Router
